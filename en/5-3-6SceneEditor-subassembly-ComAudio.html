ComAudio

Icon

 

 

Description

 

The ComAudio component contains the function of playing a mp3 or wav type file. After binding to a ComAudio component, a NULL object will be modified to a ComAudio object.

 

Properties

 

For more information, see Scene component general properties.

 

Feature

Name

 

The audio's filename，which can let us get component in programs (getComponent). The component's name can be amended and the default name is CCComAudio.

 

AudioFile

 

The path of mp3 or wav type file.

 

Loop

 

Set it to loop the audio or not.

 

PlayControl

 

  • Play：Plays audio.
  • Stop：Stops playing audio.
