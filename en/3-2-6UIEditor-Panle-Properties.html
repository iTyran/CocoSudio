Property Panel

Overview

 

The panel displays widgets' property.

 

Functionality

Search

 

1. Describes the active widget.

2. CocoStudio searches properties according to the user's input in the search field.

 

General Property

Please refer to the  UI Widget General Property Page。

 

Feature

 

Feature sets widgets' properties. Every widget will have its own unique properties. Refer to each widget's detailed page for more information.

For more information, see Widget List Page.

 

Reset

 

ight Click on the Propertie’s name, The Reset button headlined 'Reset to Defaults' will be show.If you click the Reset button，the value of this properties will be reset to defaults。

 

Note： A part of properties cannot be resetted, and the button's state is not available
