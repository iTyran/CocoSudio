Introduction

CocoStudio Introduction

CocoStudio is a free suite of mobile game development tools that helps you create awesome games.

CocoStudio contains a tool set of four editors: the user interface (UI) editor, the Animation editor, the Scene editor and the Data editor. The four editors are tailored for the roles of UI artist, animation artist, game designer and game system designer.

CocoStudio maximizes the productivity of game development and aesthetics. By using CocoStudio, game designers can focus more on game design without having to wait for the completion of the programmer's work. CocoStudio allows for more iterative cycles of game development. With more productive creative cycles, the game development will be more successful. At the same time, with separate tool sets each role can work with each tool easily.

Download CocoStudio

You can refer to the CocoStudio website at cocos2d-x.org to download the latest version, which includes the CocoStudio installation package and corresponding parsers.

Note: If you want to use the files exported by CocoStudio, you will need to use the cocos2d-x parser library. The parser is based on cocos2d-x and you must choose the correct parser version according to the CocoStudio's version requirement.

Installation

Software Prerequisites

  1. Windows XP/Windows 7/Windows 8
  2. Runtime .Net Framework 4.0 or above
  3. Visual C++ 2010 Redistributable
  4. Visual C++ 2012 Redistributable

Hardware Prerequisites

  1. A processor which can work at speeds of 2GHz or above
  2. Memory of at least 1Gb.
  3. A hard disk which stores at least 100Mb.

Installation Instructions

  1. Double-click the installation package to begin installation.
  2. Select Accept after reading the Software Usage License.
  3. Select Next to continue.
  4. Choose your installation directory.
  5. Finish installation.

Update

Update will uninstall your old copy and replace it with a new one. It will not affect your existing projects. CocoStudio will automatically check for a new version each time CocoStudio is launched.
