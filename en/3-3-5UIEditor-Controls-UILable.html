Label[UILabel]

Icon

Description

The UILabel subclass implements the text box in the touch screen. The UILabel subclass is a subclass of UIWidget.

The widget provides methods to set font size, font, text content and other attributes.

Common properties

 

For more information, see Common properties.

 

Feature

Animation

 

When activated and turned on, the widget will amplify the click effect.

 

Font size

 

Indicates the font size.

 

Font style

 

Specifies a text font, which supports Arial, bold, and Microsoft Yahei.

 

Text

 

Displays the content of the text.
