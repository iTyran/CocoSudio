UI Animation List Panel

Overview

 

This panel manages all of your UI animations.

 


Functionality

Modify animation name

 

You can double click the left mouse button on a animation name or just press F12 to activate the function of modifying animation name.

 

Context Menu

Add Animation

 

Add a new animation in the current position.

 

Copy Animation

 

Copy the selected animation and paste it at the end of animation list.

 

Delete Animation

 

Delete the selected animation.
