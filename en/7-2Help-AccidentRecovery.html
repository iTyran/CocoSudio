Backup and Restore

Backup

 

In order to protect the user's data's safety, CocoStudio will automatically create a scheduled backup. For this function, users have no need to do anything.

The backup system follows these rules:

1. Automatically save a backup every three minutes.

2. Keep the latest 10 backup files, and the former backup files will be deleted.

 

Restore

 

While editing the projects, if an accident happened and the application closed, such as the collapse of the application, system halted, power outages and so on, the user can open the original project and choose a backup in the following window:

 

 

Users can choose to restore the backup by the time depending on the users' need.

 

Note:

1. After restoring the backup, changes cannot be cancelled.

2. It is strongly recommended that users copy the project to avoid restoring unnecessary files.
