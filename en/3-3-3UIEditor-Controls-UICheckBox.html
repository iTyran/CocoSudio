CheckBox[UICheckBox]

Icon

Description

UIScrollView is a subclass of UIWidget that implements a check box in the touch screen.

The widget provides methods to set the different states of the background image and checks the selected tag of the state image and other attributes. By using the set method, you can specify a different apperance for the each checkbox state.
