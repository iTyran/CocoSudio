ComAttribute

Icon

 

 

Description

 

The ComAttribute component contains the function of rendering a UI component. After binding a ComAttribute component, a NULL object will be modified to a ComAttribute object.

 

Properties

 

For more information, see Scene Component general properties.

 

特性

Name

 

The Component's name, which can let us get component in programs(getComponent). The Component's name can be amended and the default name is CCComAttribute.

 

File

 

The path of a json date file.
