Loading bar[UILoadingBar]

Icon

Description

The UILoadingBar subclass implements the progress bar of the touch screen. UILoadingBar is a subclass of UIWidget.

The widget provides methods to set the value for the progress bar, progress bar orientation, image resources and other attributes.

Feature

Progress

 

Sets the widget to the default status and ranges between 0 and 100.

 

Type

 

Sets the progress bar orientation to either Left_TO_Right (from left to right) or Right_To_Left (from right to left). The progress bar orientation is set to Left_To_Right by default.

 

Image resources

 

Configures the image resources (png, jpg) and controls the default display.
