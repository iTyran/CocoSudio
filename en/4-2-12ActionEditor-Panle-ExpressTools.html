Functions

Physical/Animation Mode Switch

     

 

Switches between Physical Mode and Animation Mode.

 

Rotate counterclockwise by 90 degrees

 

 

Rotates the selected widget counterclockwise by 90 degrees.

 

Rotate 90 degrees

 

 

Rotates the selected widget 90 degrees.

 

Align center point

 

 

Aligns all selected widgets' center points. You should select more than one widget.

 

Rotate Skeleton Mode

 

 

The result of the skeleton in the rendering area after rotating.

Move Skeleton Mode

 

 

The result of the skeleton in the rendering area after moving.

 

Zooming Mode

 

 

The result of the skeleton in the rendering area after zooming.

 

Display Skeleton Mode

 

Displays the skeleton when the object is selected, otherwise the skeleton will not display. This switch just appears in Animation Mode.

 

Create Skeleton Mode

 

 

Creates the skeleton when the object is selected; otherwise the skeleton will not be created. This switch just appears in Physical Mode.

 

 

The two switches of display and create skeleton can exist in the same position in the application, but appear in different situations.
