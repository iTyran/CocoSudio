Menu Panle

Introduction

 

Editor menu bar is a collection of common operations that include four categories: File, Edit, View, Help.


Feature

File(Alt+F)

New Project(Ctrl+N)

 

If you have been opened a project When Creating a new Scene project, ediror will prompte to save the current project.

 

Note:

1: The file name can not contain the following symbols: "？"、"、"、"╲"、"*"、"""、"""、"<"、">"、"|".

2: Do not create a project on the desktop if you are using Mircosoft XP systems.

 

Open Project(Ctrl+O)

 

Open an existing project.

 

Save Project(Ctrl+S)

 

Save the project which is currently being edited.

 

Save As(Ctrl+Shift+N)

 

 

The current project is saved to a new disk path.

 

Note: The file name can not contain the following symbols: "？"、"、"、"╲"、"*"、"""、"""、"<"、">"、"|".

 

Export Project

 

Export the selected cancas as project resources.

 

Import Resources

Importing Files

1. Supports multi-select Import.

2. Files with duplicate names will not be imported.

 

Import Folder

1. Select folder from disk to import for use in CocoStudio.

2. Folder with duplicate names will not be imported.

 

Exit(Alt+F4)

 

If the project has been modified, the Editor will remind the user to save the project.

 

Edit(Alt+E)

Undo(Ctrl+Z)

 

To reverse the last action.

 

Redo(Ctrl+Y)

 

Recovery the reversed action.

 

View(Alt+V)

Object Struct

 

Open Object Struct window.

 

Properties

 

Open Properties window.

 

Resources

 

Open Resources window.

 

Output

 

Open Output window.

 

Start Page

 

Open Start Page window.

 

Reset Window Layout

 

Reset the editor to the initial layout.

 

Help(Alt+H)

Open help documents. (F1)

 

Open help online documents。

 

About Cocostudio

 

Open cocostudio information window。

 

Check New Version

 

Check update。

 

Language(Alt+L)

 

Take effect after restarting the editor if you changed the language.
