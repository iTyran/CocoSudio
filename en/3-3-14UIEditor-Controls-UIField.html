Field[UIField]

Icon

Description

UIField is a subclass of UIWidget that implements the field in the touch screen.

The widget provides methods to set font size, font, password mode, length, text, placeholder text and other attributes.

Feature

font size

 

Sets the font size.

 

font

 

Now supports only Arial, bold, Microsoft yahei.

 

Open password

 

1. Set whether to open the password field style.

2. Character style: set a password character display style.

3. character style input limit: only ASCII(33-126) for a single character.

 

Length limit

 

1. Set whether to open the filed text length limit.

2. Set the maximum length of the filed.


Text

 

Display text.

 

Placeholder

 

Set opening the filed text length limit or not.
