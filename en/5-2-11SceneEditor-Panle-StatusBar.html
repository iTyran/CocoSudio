Status Bar

Introduction

The status bar displays the currently selected control's properties as well as the rendering area scaling factor adjustment.

Functions

name

Rotation degree

y-axis scaling factor
Render Area scaling slider
Render area scaling values input box
Drop-down box to quickly select the scaling factor
Restores the default scaling factor

To adjust the scaling factor:

  1. Drag the slider.
  2. Directly input the scaling factor into the input box.
  3. Select the value in the drop-down box.

Restore the default scaling factor. Select the Restore button to restore to the default scaling factor.
